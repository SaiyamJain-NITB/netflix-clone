# Netflix Clone

A front-end clone of the Netflix landing and browse pages, built with HTML, CSS, and vanilla JavaScript.

## Features
- Responsive layout modeled on Netflix's landing and browse pages
- Content rows with hover-to-preview on movie/show cards
- Static content (no backend or external API — all movie/show data is hardcoded)

## Built with
- HTML
- CSS
- JavaScript

## Running it locally
1. Clone or download this repo
2. Open `index.html` in your browser

No build steps or installs needed — it's a plain HTML/CSS/JS project.

This is a front-end practice/portfolio project built for learning purposes and is not affiliated with or endorsed by Netflix.
